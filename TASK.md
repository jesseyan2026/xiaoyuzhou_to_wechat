# 任务需求文档

## 当前任务

### 任务描述
将小宇宙播客博客内容转换为微信公众号文章格式。

### 核心需求
1**完整转述博客内容** - 基于播客内容重新组织成流畅的文章，要求故事性强，而非简单罗列；shownotes作为一定参考，更丰富的内容是
博客里面的语音内容部分；
2**以第三人称为叙述者** - 使用第三人称转述播客内容，避免使用第一称或第二称。
3**输出为微信公众号文章格式** - 生成的文章需要符合微信公众号的文章格式要求，包括标题、作者、内容等。
4**文章风格** - 文章风格为故事/叙述风格，首行缩进、较大行距。

### 处理流程
1. 抓取小宇宙博客页面内容
2. 以第三人称为叙述者，转述播客中的核心观点、故事和干货
3. 输出为微信公众号文章格式（HTML）

### 输出要求
- 文章风格：故事/叙述风格（首行缩进、较大行距）
- 作者署名："我的朋友"
- 格式：微信公众号兼容的 HTML
