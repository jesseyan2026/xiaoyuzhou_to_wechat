# 播客智能转录工具使用指南

## 功能说明

这个工具实现了**智能转录流程**，优先从外部信息源获取文稿，节省时间和token：

### 智能流程
```
1. 分析播客页面 → 提取标题、嘉宾、关键话题
2. 搜索外部信息源 → 微信公众号、今日头条、Bing搜索
3. 如果找到文稿 → 直接抓取使用 ✅
4. 如果没找到 → 下载音频 + Whisper转录 🎵
5. 如果音频也没有 → 提示手动方案 💡
```

## 环境准备

### 1. 安装系统依赖

**macOS:**
```bash
brew install ffmpeg
```

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install ffmpeg
```

### 2. 安装 Python 依赖

```bash
# 基础依赖（必需）
pip install requests beautifulsoup4 lxml

# 音频转录依赖（可选，只有需要转录音频时才安装）
pip install openai-whisper torch
```

## 使用方法

### 基本用法（智能模式）

```bash
python transcribe_podcast.py <小宇宙播客链接>
```

示例：
```bash
python transcribe_podcast.py https://www.xiaoyuzhoufm.com/episode/69d7f994b977fb2c4789b1ac
```

### 参数选项

```bash
# 跳过外部搜索，直接转录音频
python transcribe_podcast.py <URL> --skip-search

# 指定输出目录
python transcribe_podcast.py <URL> --output ./my_transcriptions

# 使用更大的Whisper模型（更准确但更慢）
python transcribe_podcast.py <URL> --model large

# 使用 OpenAI API 转录（需要 API Key）
python transcribe_podcast.py <URL> --use-api --api-key your_api_key
```

### 模型大小选择（仅音频转录时有效）

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| tiny | 最快，准确度最低 | 快速测试 |
| base | 较快，准确度一般 | 简短内容 |
| small | 平衡选择 | 一般用途 |
| medium | **推荐**，准确度好 | 日常使用（默认） |
| large | 最准确，速度慢 | 高质量要求 |

## 工作流程详解

### 第1步：提取播客信息

工具会自动从小宇宙页面提取：
- 播客标题
- 主播/作者
- 嘉宾名字（智能识别）
- 关键话题（如出海、搞钱、AI等）
- 音频链接（如果可访问）

### 第2步：搜索外部信息源

工具会搜索以下平台：

| 平台 | 说明 |
|------|------|
| **微信公众号** | 优先搜索，质量通常较高 |
| **今日头条** | 补充搜索 |
| **Bing搜索** | 综合搜索其他来源 |

搜索关键词自动生成：
- 完整标题
- 标题 + 嘉宾名字
- 关键话题组合
- 嘉宾 + 播客名

### 第3步：抓取文章内容

找到相关文章后，工具会尝试抓取：
- 微信公众号文章（通过搜狗搜索）
- 其他网页文章

### 第4步：备用方案（音频转录）

如果外部搜索无结果，自动切换到音频转录：
1. 下载音频文件
2. 使用 Whisper 转录
3. 生成文字稿

## 已知限制

### 外部搜索的限制

1. **微信公众号**：
   - 需要通过搜狗搜索
   - 部分文章可能已删除或无法访问
   - 抓取可能受反爬限制

2. **音频访问限制**：
   - 小宇宙音频可能需要登录
   - 可能返回 403/401 错误

## 常见问题

### Q: 为什么优先搜索外部文稿而不是直接转录？

**A**: 有以下几个原因：
1. **节省时间**：搜索只需要几秒，转录需要几分钟到几十分钟
2. **节省Token**：音频转录成本高（约 $0.006/分钟）
3. **准确度高**：外部文稿通常是人工整理或官方发布，准确度更高
4. **结构清晰**：外部文章通常有更好的段落结构

### Q: 外部搜索能找到所有播客的文稿吗？

**A**: 不能。只有热门播客或商业活动（如听友会）才可能有外部文稿。对于普通单集，可能需要音频转录。

### Q: 如何判断找到的文章是否准确？

**A**: 工具会显示：
- 文章来源（微信公众号优先）
- 文章标题
- 内容摘要

您可以查看这些信息判断相关性。

### Q: 转录质量如何？

**A**: Whisper 对中文支持较好，但可能有：
- 专业术语识别错误
- 多人对话区分不清
- 口音较重时准确度下降

建议转录后进行人工校对。

### Q: 能否使用 GPU 加速转录？

**A**: 可以。安装 CUDA 版本的 PyTorch：
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

1. **版权**：转录仅供个人学习使用，请勿侵犯版权
